# HTML5基础详解

### 1.声明<!DOCTYPE>

Intellij IDEA复制当前行到下一行`CTRL + D`

​	HTML5也有很多个不同的版本，只有完全明白页面中使用的确切的HTML版本，浏览器才能够完全正确地显示出HTML页面。这也是<!DOCTYPE>的用处。

​	HTML5:

​	`<!DOCTYPE html>`

### 2.HTML标题

从h1到h6

```html
<body>
    hello world
    <h1>标题1</h1>
    <h2>标题2</h2>
    <h3>标题3</h3>
    <h4>标题4</h4>
    <h5>标题5</h5>
    <h6>标题6</h6>
</body>
```

### 3.HTML段落

<p></p>p标签定义段落

```html
<p>hello 蓝云甫</p>
```

### 4.HTML超链接

<a>标签定义链接

```html
<a href="http://www.baidu.com">百度一下</a>
```

### 5.HTML图像

<img>标签定义图片
